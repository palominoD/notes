# NOtas de Actiuve directory domain services
* formato de base de datos distribuidas que permite almacenar informacion de configuraciones, permisos y usuarios.
* formato usado por exchange de MS
* administrar la infraestructura de red
* basada en contenedores (bosques, dominio u organizacion)
* sistema que permite el sso
* centralizar objetos y politicas
* utiliza dns, kerberso, dhcp y mas

## termnologia y jerarquia
* DC = dopmain contoler
* objeto = usuartio, contactos, grupos, carpeas compartidas e impresoras, edpoits, dc y ou)
* dominio = collecion de bjetos (puede tener subdominios)
* arbol = coleccion de dominio
* bosques = coleccion de arboles
* atributo = caracteriticas de objetso editables (nombre, edad, etc)
* esquema = conjunto de atributos (editables); define el modelo de cada objeto (similar a las clases) 
* catalogo global = almacen de informacin dada a los atributos de manera parcial para indexar cada atributo de esquema
